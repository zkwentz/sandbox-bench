Scoring Methodology

Each provider is scored from 0-100 based on six weighted metrics. The benchmark measures the full lifecycle of an AI agent interacting with a sandbox: authenticate, create, execute code, read/write files, and destroy.

Time 30%
Seconds from API key to working sandbox
Errors 20%
Errors encountered during onboarding
Tool Calls 15%
Number of API/SDK calls required
Friction 15%
Manual steps or workarounds needed
Cost 10%
USD cost per benchmark run
Discoverability 10%
Docs, MCP server, OpenAPI, llms.txt

Grades: A (85-100) · B (70-84) · C (55-69) · D (40-54) · F (0-39)
